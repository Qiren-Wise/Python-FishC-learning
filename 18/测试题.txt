0、你有听说过DRY吗？
答：没有
修改：DRY是程序员们公认的指导原则：Don’t Repeat Yourself.
快快武装你的思维吧，拿起函数，不要再去重复拷贝一段代码了！

1、都是重复一段代码，为什么我要使用函数（而不使用简单的拷贝黏贴）呢？
答：效率更高，特别是数据量大的时候。
修改：使用函数:
        0) 可以降低代码量（调用函数只需要一行，而拷贝黏贴需要N倍代码）
        1) 可以降低维护成本（函数只需修改def部分内容，而拷贝黏贴则需要每一处出现的地方都作修改）
        2) 使程序更容易阅读（没有人会希望看到一个程序重复一万行“I love FishC.com”）

2、函数可以有多个参数吗？
答：可以，想添加多少添加多少。

3、创建函数使用什么关键字，要注意什么？
答：应该要与用途能搭上边的字母来当关键字，必要时候还需要用函数文档来解释。
修改：使用“def”关键字，要注意函数名后边要加上小括号“()”，然后小括号后边是冒号“:”，然后缩进部分均属于函数体的内容。

4、请问这个函数有多少个参数？
>>> def MyFun((x, y), (a, b)):
        return x * y - a * b

答：这是一个无效语法.....
修改：如果你回答两个，那么恭喜你错啦，答案是0，因为类似于这样的写法是错误的！
      我们分析下，函数的参数需要的是变量，而这里你试图用“元祖”的形式来传递是不可行的。
      我想你如果这么写，你应该是要表达这么个意思：
>>> def MyFun(x, y):
        return x[0] * x[1] - y[0] * y[1]

>>> MyFun((3, 4), (1, 2))
10

5、请问调用以下这个函数会打印什么内容？
>>> def hello():
        print('Hello World!')
        return
        print('Welcome To FishC.com!')
答：会打印'Hello World!'，因为返回值不要print来指定，直接在return后面加''在里面输入值就好了。
当Python执行到return语句的时候，Python认为函数到此结束，需要返回了（尽管没有任何返回值）。


总结：这节课我们了解了函数的定义与返回值。
函数使用def来进行定义，后面跟着函数名和参数，如def liu():
下面可以填函数的内容，如果要调用函数直接输入函数名加()就好了，如果要多次调用可以用for循环。
括号里面就是填函数的参数，这样可以使每次调用函数有不同的实现。如def liu(x):,后面可以函数的内容+参数，这样就会两者一起调用了。
参数也可以定义多个，只要中间用逗号隔开即可。
函数的返回值用return来表示，后面就是跟着需要返回的值，return和print有许多相似的。如果不设定，默认返回none。
有多个返回值的时候默认用元组来进行打包，当然也可以你先用列表打包起来在返回。
