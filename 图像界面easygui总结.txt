总结：这节课了解图像界面的使用
一、导入模块
    一共有三种方法：
    //方法一：
    import easygui
    easygui.msgbox('...')

    //方法二：
    from easygui import *   # 使用通配符导入所有easygui模块
    msgbox('...')   # 这样更容易调用函数

    //方法三：
    import easygui as g     # 将模块导入到g当中，之后我们就可以之间使用g来操作模块了
    g.msgbox('...')
    我们还可以在idle上调用egdemo()来查看easygui拥有的各种功能，如：
    >>> import easyguilieb
    >>> easygui.egdemo()

二、EasyGUI 函数的默认参数
对于所有对话框而言，前两个参数都是消息主题和对话框标题
    - 标题默认时空字符串
    - 两个参数同时设置，如：>>> msgbox('我爱小甲鱼^_^', '鱼油心声')

三、使用关键字调用函数
调用easygui中的函数的时候，我们也可以和其他函数一样使用关键字定义，如：easygui.choicebox(title='你好',mag='请选择',choices = ['one','two'])
但是要和普通函数一样，注意关键字参数和位置参数不能搞混了，具体可复习书本P67

四、按钮控件
1. msgbox()
    这个提示框只有一个ok按钮，我们可以通过在函数的第三个位置，或者利用关键字参数来改变按钮名称
    如：msgbox('good','hello',ok_button='完成')
2. ccbox()
    这个提示框有两个按钮，我们可以在函数的第三个位置，或者关键字参数来定义，不过上方不同，这边的关键字参数是choices
    如：ccbox('明天可以来我家玩吗','邀请',choices=['yes','no']) 
3. ynbox()
    和ccbox()基本相同，只不过这里的按钮choices的值不同而已，ccbox()默认continue/cancel，这个默认yes/no，并且默认F1和F2的快捷键
4. buttonbox()
    这个可以自己定义多个按钮，默认定义三个按钮，但是位置参数的前两个还是msg和title，第三个位置参数才是按钮，同样关键字是choices
    如：buttonbox('明天做什么','选择',choices=['上班','学习','打游戏']) //这里的多个按钮同样需要用列表包含
    # 当用户单击任一按钮的时候，函数返回该按钮的值，如果点击右上角的'X'，默认返回第一个按钮的值
5. indexbox()
    和buttonbox()基本相同，区别就是上一个返回按钮的值，这个返回按钮的索引值，从0开始索引，默认两个按钮
    如：indexbox(choices=['one','two','three'])     //点击第一个按钮返回0，以此类推
6. boolbox()
    和ccbox()、ynbox()基本相同，不过这里的两个按钮默认是true和false，并且返回的值也是true和false
//我们还可以设置快捷键噢，如：choices=['y[e]s','[n]o']，这样我们敲击键盘上的e或n就分别代表yes/no
    还有一种方法，如：choices=['[<F1>]yes','[<F2>]no']，这样我们敲击键盘上的F1或F2就分别代表yes/no

五、控件中显示图片
当你调用一个含按钮的空间时，例如上面(四)说的那些空间，可以为image赋值，可以设置.gif格式或者.png格式的图片，会动的gif好像不行
如：msgbox(image='D:\学习\树洞网站\images\树洞.png')，这样就会显示出该路径下的图片了，我们还可以点击该图片，这样就会返回你输入的路径名

六、选择控件
1. choicebox()
	此函数为用户提供了一个简单的可选择列表，在括号内的第三个位置，或者用关键字参数choices可以来设定此列表，这些选项会按照列表索引进行排序
		另外还可以使用键盘来控制选项，但是只有纯英文的选项支持查找，利用首字母索引
			- 假设选项中有一个'baba'的选项，那么单击键盘上的'b'键，即可选择此选项
			- 如果选项中有多个'b'打头的选项，则会从你当前选择的位置依序往下一个个查找，如果还未进行选择，默认选择列表第一个'b'打头的选项
			- 如果查询不到'b'打头的选项，则会顺延至'b'前一个字母'a'查询，如果查询至'a'还找不到，默认选择列表第一项
2. multchoicebox()
	此函数也是提供一个可选择的列表，不管可以支持用户选择0个或者多个选项，这个控件的按钮有四个，分别是'Cancel':取消，'Select All':全选，'Clear All':取消当前选中项，'OK':完成
	
七、输入控件
1. enterbox()
	为用户提供一个简单的输入框，前两个位置参数默认还是'msg'和'title'，并且自带两个按钮，分别是'OK','Cancel'，单击'OK'，返回用户输入的内容
		如果你设置的消息中有必须要保留的首/尾空格，可以设置产生strip=False 保留
2. integerbox()
	为用户提供一个可输入整型数值的输入框，用户只能输入一定范围内的值(lowerbound参数获取最小值，upperbound参数获取最大值，不设置默认范围1-100)
		如果不输入设定范围内的值，会要求重新输入
//我们还可以通过设置default='默认输入内容'来定义用户未输入时，默认输入的内容噢！
3. multenterbox()
	为用户提供多个简单的输入框，同样前两个默认参数时'msg'和'title'
		第三个参数是fields，这个参数用于设置输入框左边的词，第四个参数是value，用于设置默认输入值，三四个参数都是要求输入一个序列噢
			注意：
				- 如果用户输入值比选项少的话，未输入的选项默认返回空字符串
				- 如果用户输入值比选项多的话，则返回的列表中的值将截断为选项的数量。
				- 如果用户选择'OK'，会返回一个列表，列表中的每一项按顺序对应选项中输入的值；选择'Cancel'或者'X'，默认返回none值

八、密码控件
//有时候我们需要用户输入一些敏感信息，那么界面中的字符是不应该显示出来的，如："******"
1. passwordbox()
	此控件和enterbox()一样，不同的是用户输入的内容用'*'显示出来，不过只有输入时是以加密的形式显示，单击'OK'是还是以明文的方式返回字符串
2. multpasswordbox()
	此控件和multenterbox()差不多，同样用fields表示输入框左边的字符，不过不同的是，输入框的最后一行是'*'加密的，其他行则是明文正常显示

九、文本控件
1. textbox()
	为用户提供一个文本输入框，除了正常的'msg'和'title'
		我们还可以设置text参数来设置默认显示的文本(可以是字符串，列表，元组)
		codebox来设置是否自动换行(此参数为true是不会自动换行，即字符单行输入可超过当前文本框宽度，超过时自动添加滚动条，默认false，自动换行，即不会超过文本框宽度)
2. codebox()
	相当于textbox中codebox=True 时的结果，不自动换行
		注：等宽字体很丑的，但适合代码编写
		
十、目录与文件
1. diropenbox()
	提供一个对话框，你可以在这个对话框中选择路径，返回用户选择的目录名(带完整路径的文件夹)，如果用户取消则返回none
		其中第三个参数为default用来设置默认打开的路径(请确保设置的默认路径存在！)
2. fileopenbox()
	和上面的大致相同，不过这个函数用于选择文件，选择文件后单击确定会返回完整的文件路径，取消则返回none
		同样有default参数，这里的设置方法有所不同：
			- default指定了一个默认路径，通常包含一个或多个通配符，使用通配符则表示显示当前文件夹下所有文件
			- default默认参数是'*'，匹配所有目标格式，如果不设置，则显示默认的路径和格式，但是我们可以通过通配符灵活的查找文件格式
			如：
			- default = 'D:\学习\*'	即显示D盘 -> 学习目录中的所有文件
			- default = 'D:\学习\*.py'	即显示D盘 -> 学习目录中的 -> 所有'.py'格式的文件
			- default = 'D:\学习\职业*.pptx'		即显示D盘 -> 学习目录中的 -> 所有以'职业'开头的'.pptx'文件
		还有第四个参数filetype，它的设置方法如下：
			- 可以是包含文件掩码的字符串列表，例如：filetypes = ["*.txt"]
			- 可以是字符串列表，列表的最后一项字符串是文件类型的描述，例如：filetypes = ["*.css", ["*.htm", "*.html", "HTML files"]]
		还有最后一个multiple参数，这个参数如果为True则可以选择多个文件，单击确定后返回的是多个文件路径组成的列表
3. filesavebox()
	提供一个对话框，让用户选择文件需要保存的路径，如果用户选择确定，返回保存的该文件的完整路径
	default参数应该包含一个文件名噢~，当然也可以设置为空，只不过要包含一个通配符
	filetypes 参数的设置方法请参考 fileopenbox() 函数。

1. exceptionbox()
	使用easygui写入程序，难免有时会出现异常，我们可以通过这个函数进行处理
	如：
		try:
			int('abc')
		except Exctption as reason:
			exceptionbox(str(reason),'异常问题')
		//出现异常后，将会弹出一个文本框，你还是可以将问题原因传到reason中赋值给控件的msg语句，并且第二个参数还是title
