总结：这节课我们了解了一些python的模块
其实我们每次写的源代码文件(.py)都是一个模块，并且python自带许多模块，掌握它们，做事将会事半功倍
前面学过的randint()就是属于random模块的，这个可以用来产生随机数，不过在用之前需要用import来引入random模块
1. OS模块
这是一个功能十分强大的模块，里面有许多函数，常用函数可以看书本P110页
- getcwd()
    # 这个函数可以帮助我们查看当前的工作目录
- chdir()
    # 这个可以用来改变当前工作目录
- listdir()
    # 这个可以把当前目录下的各个文件和文件夹列举出来，当然，你可以在括号内填入指定的文件夹，然后查询里面的文件和子文件夹
- mkdir()
    # 创建单层目录，如果该文件夹已存在，会抛出异常
- makedirs()
    # 创建多层目录，如果该多层目录存在，也会抛出异常，不过如'C:\\a\\b\\c'和'C:\\a\\b\\d'是不同的目录哦！
- remove()
    # 这个用来删除文件
- rmdir()
    # 这个用来删除目录，但是该目录里面一定要没有文件了，不然就会抛出异常
- removedirs()
    # 这个用来删除多层目录，同样要确保多层目录中无文件！
- rename(old，new)
    # 用来重命名文件，函数中有两个参数，第一个填旧文件名，第二个填需要替换的新文件名
- system()
    # 这个可以用来调用一些系统工具，如括号内填"cmd"，就调用命令提示符，"calc"，就会调用计算器。和键盘上win+r后输入的同一个效果
- walk(top)
    # 这个函数可以遍历top参数下的所有文件/文件夹，并返回一个三元组[路径,[包含目录],[包含文件]],具体解释看书本P113

2. OS.path模块
这个也是一个模块，用来完成一些正对路径名的一些操作，常用函数可看书本P113
- basename()
    # 去掉目录名，单独返回文件名，括号内填路径
- dirname()
    # 去掉文件名，单独返回目录路径，括号内填路径
- join(path1[,path2])
    # 将path1和path2合并成一个路径，当然还可以有更多参数组合在一起，如果最开始的第一个参数仅有盘符的话，需要加上双斜杠，如join('C:\\','a','b')，这样才行
- split() 和 splitext()
    # 这两个函数都用于分割路径，前者将路径和文件名分离，后者将路径文件名与后缀名分离，可以用来读取后缀
- getsize()
    # 这个可以获得文件的尺寸，以字节为单位，大概也就是多少个字符
- getatime() 、 gatctime() 、 getmtime()
    # 这三者分别用来读取文件的最近访问时间、创建时间和修改时间，返回值是浮点数，如果要换算可以用time模块的gmtime()或localtime()，具体可看书本P115
